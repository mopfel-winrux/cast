# Cast Features

## Tier 0 — MVP
- Subscribe to podcast by RSS URL
- List subscribed podcasts
- List episodes for a podcast
- Play episode (stream from source URL)
- Basic web UI (podcast grid → episode list → player)

## Tier 1 — Core Experience
- Save/resume playback position
- Episode queue (add, remove, reorder)
- Playback speed control (0.5x–3x)
- Mark episodes played/unplayed
- Auto-refresh feeds on timer
- Podcast artwork display

## Tier 2 — Enhanced
- Download episodes to local cache
- Podcast search/discovery (PodcastIndex API)
- OPML import/export
- Episode filtering (played, unplayed, in-progress)

## Tier 3 — Advanced
- Cross-ship sync (share subscriptions)
- Episode notes/bookmarks
- Listening statistics
- New episode notifications
- PWA manifest for mobile install
